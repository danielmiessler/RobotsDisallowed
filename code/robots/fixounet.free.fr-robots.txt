         <tr><td style="padding: 2px; text-align: center; font-weight: bold; color: #61718A">ERREUR 404 - Document non trouv&eacute;</td></tr>
         <tr><td style="padding: 2px; text-align: left">
         <br/>
         <br/>L'erreur 404 est un code d'erreur dans le protocole HTTP. Ce code est renvoy&eacute; par un serveur HTTP pour indiquer que la ressource demand&eacute;e (g&eacute;n&eacute;ralement une page web) n'existe pas. 
         <br/>Certains navigateurs web affichent alors le message &laquo; 404 File Not Found &raquo; (de l'anglais signifiant &laquo; fichier non trouv&eacute; &raquo;) &agrave; destination de l'internaute.
         <br/>
         <br/>Le premier 4 indique une erreur dans la requ&ecirc;te, ici une mauvaise URL, venant d'une page obsol&agrave;te ou d'une erreur de saisie d'adresse web de la part du visiteur. Le dernier 4 indique le probl&egrave;me caus&eacute; par cette erreur : la ressource est introuvable.
         <br/>(<a href="http://fr.wikipedia.org/wiki/Erreur_HTTP_404" target="_blank">http://fr.wikipedia.org/wiki/Erreur_HTTP_404</a>)
         <br/>(English version: <a href="http://en.wikipedia.org/wiki/HTTP_404" target="_blank">http://en.wikipedia.org/wiki/HTTP_404</a>)
